Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The ul element stands for unordered lists and goes with li elements to create bullet points

CSS

  • Css stands for cascading style sheets, and helps add the styling for the webpage

Git

  • Git is a version control system that tracks changes in any set of computer files, used for coordinating work among programmers collaboratively developing source code during software development.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is how the user is able to interact with the webpage
